1. Why we use React?
    
React is a powerful JavaScript library used to build fast and interactive user interfaces, especially for single-page applications. It follows a component-based architecture, allowing developers to create reusable UI parts that make code easier to manage. React uses a Virtual DOM to efficiently update only the parts of a webpage that change, improving performance. With its declarative syntax, developers can describe what the UI should look like, and React handles the rendering automatically. Backed by Meta (Facebook) and a strong community, React offers great flexibility, reusability, and performance, making it one of the most popular choices for modern web development.

6. what is babel compiler?

Babel is a JavaScript compiler that helps your code run on all browsers.

It takes modern JavaScript (ES6 and above) or JSX (used in React) and converts it into older JavaScript that browsers can understand.
